# Python手写数字识别系统 python601

## 项目概述

本项目是基于Python的手写数字识别系统，集成了登录、首页、手写字识别、管理、改密码、个人信息、用户管理等多个功能模块。

## 技术栈

- **开发工具**：IDEA、Eclipse
- **编程语言**：Python
- **支持数据库**：MySQL 5.7+
- **框架**：B/S、Django

## 功能模块

- 登录
- 首页
- 手写字识别
- 管理
- 改密码
- 个人信息
- 用户管理

## 系统角色

- 普通用户：可以进行手写数字识别、查看个人信息等操作
- 管理员：可以进行用户管理、系统设置等操作

## 运行环境

- Python环境
- MySQL 5.7+数据库
- 适用于常见Web服务器

## 部署步骤

1. 安装Python环境
2. 安装并配置MySQL数据库
3. 部署Web服务器
4. 导入数据库文件
5. 运行系统
